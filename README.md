Star method of answering behavioural questions =>

STAR - [Situation, Task, Action, Result]

eg: How you improved performance of your application in your last project?

[Situation]: In my previous project HIG(Hartford Insurence Group), production APIs were experiencing frequesnt timeoutsduring peak traffic.
[Task]: I was responsible for identifying root cause and improving system's performance.
[Action]: I analyzed thread dumps, heap dumps, optimized DB queries, implemented coding and for fault tolerence Circuit breakers implemented.
[Result]: As a result the response time improved by 45% and production incidents were reduced by 60%.
